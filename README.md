# 🚀 Abhay Kumar Singh - Portfolio

Welcome to my personal portfolio website! This project showcases my skills, experience, and projects as a Cyber Security Analyst and Cloud Developer.

## 🌐 Live Demo
[View Live Portfolio](https://abhay52004.github.io/project5/) (Requires GitHub Pages to be enabled)

## ✨ Features
- **Responsive Design**: Fully responsive across all devices (Mobile, Tablet, Desktop).
- **Modern UI/UX**: Built with a clean, dark-themed glassmorphism aesthetic.
- **Animations**: Uses AOS (Animate on Scroll) for a dynamic user experience.
- **Sections**:
  - **Hero**: Introduction with a typewriter effect.
  - **About Me**: Professional background and education details.
  - **Skills**: Technical expertise in Security & Analysis, Security Tools, and Infrastructure.
  - **Projects**: Showcase of recent work.
  - **Contact**: Reach out through email or social media.

## 🛠️ Tech Stack
- **Frontend**: HTML5, CSS3, Bootstrap 5, JavaScript
- **Fonts & Icons**: Google Fonts (Outfit), Font Awesome 6
- **Animations**: AOS Library
